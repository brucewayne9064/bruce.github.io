1.概述

优先队列PriorityQueue是Queue接口的实现，底层是通过二叉堆实现的。队列首部存放的是最大或者最小的元素。并不遵循FIFO原则。

PriorityQueue可以对其中的元素进行排序，可以存放基本数据类型或者自定义类型，对于基本数据类型，默认排序是从队首开始升序排序（小顶堆）；对于自定义类型，需要自定义比较器。
